В QUIK функция ParamRequest используется для запроса параметров инструмента из базы данных. Параметр db_name определяет, какой именно параметр вы хотите получить. Вот список допустимых значений для db_name:

Основные параметры:
"LAST" — последняя цена сделки.

"BID" — лучшая цена покупки (лучший бид).

"OFFER" — лучшая цена продажи (лучший аск).

"SPREAD" — спред между лучшим бидом и аском.

"BIDDEPTH" — глубина стакана по лучшему биду.

"OFFERDEPTH" — глубина стакана по лучшему аску.

"VOLUME" — объём торгов за текущий день.

"OPEN" — цена открытия.

"HIGH" — максимальная цена за день.

"LOW" — минимальная цена за день.

"CLOSE" — цена закрытия предыдущего дня.

"LASTCHANGE" — изменение последней цены относительно предыдущей сделки.

"LASTCHANGEPRCNT" — изменение последней цены в процентах.

"QTY" — количество лотов в последней сделке.

"TIME" — время последней сделки.

"VALUE" — оборот в денежном выражении за текущий день.

"WAPRICE" — средневзвешенная цена.

"NUMTRADES" — количество сделок за текущий день.

Параметры для фьючерсов:
"SETTLEPRICE" — расчётная цена.

"OPENPOSITION" — открытый интерес.

"SETTLEDATE" — дата исполнения.

Параметры для опционов:
"OPTIONTYPE" — тип опциона (Call/Put).

"STRIKE" — страйк-цена.

"EXPIRATION" — дата экспирации.
